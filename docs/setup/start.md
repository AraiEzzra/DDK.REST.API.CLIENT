# Start

1. Open docker folder

```bash
cd docker
```

2. Starting DDK Rest API client uses docker-compose

```bash
docker-compose up -d
```

or use `start.sh` script

```bash
sh start.sh
```

# Stop

To stop the container, open docker folder and enter

```bash
docker-compose down
```

or use `stop.sh` script

```bash
sh start.sh
```
